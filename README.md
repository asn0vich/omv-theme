# omv-theme

With this tool, you can easily switch between the GUI theme

To install:         
git clone https://github.com/virgil-av/omv-theme.git && cd omv-theme && chmod +x install-omv-theme.sh && ./install-omv-theme.sh && cd

To uninstall:       
rm -r omv-theme                                                                               
rm -r /usr/bin/omv-theme

To Start:                                       
omv-theme
